# Javascript Array Exercise: String Manipulation

## Description
Practice with arrays and string manipulation: 
* reverse order
* alphabetize order
* determining palindromes

## How to run
In your terminal run:
```
http-server -p 9999
```
then navigate to [here](http://localhost:9999) in your browser.
